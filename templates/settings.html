How to get Gmail App Password:
  1. Go to myaccount.google.com
  2. Click Security
  3. Turn on 2-Step Verification if not already on
  4. Search "App Passwords" in the search bar
  5. Select app: Mail → device: Windows Computer
  6. Click Generate → copy the 16-character password
  7. Paste it above (spaces are fine)
